Install
------------------------------------------------------------------------------

``airflow_dag_artifact`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install airflow-dag-artifact

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade airflow-dag-artifact
